== Description ==
Wuqi is a clean and simple theme for blogging and writing. It features bold typography and room for your featured images. You can pick between a two-column or a one-column layout. Individual post and pages layouts are also possible. It also comes with custom header support.

Learn more at [https://presscargo.io/themes/wuqi](https://presscargo.io/themes/wuqi).

== Installation ==

= From within WordPress =
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in "Wuqi" in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
